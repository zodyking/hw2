## Home Weather

A Home Assistant custom integration with a full-screen weather panel, animated conditions, NWS alerts, hurricane tracking, and configurable TTS.

### Features

- Live conditions with animated atmosphere hero
- 7-day and 24-hour forecasts
- NWS alerts with expandable details
- Hurricane tracker with forecast cone and home threat summary
- Space weather monitoring
- Modular card layout (customizable dashboard)
- True light/dark theme support
- Mobile-first responsive design

### Requirements

- Home Assistant 2024.1+
- Weather entity supporting weather.get_forecasts
